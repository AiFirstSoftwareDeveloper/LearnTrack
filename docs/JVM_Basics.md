# What is bytecode
### Bytecode is an intermediate, platform-independent code generated by the Java compiler (javac) from Java source files (.java) and stored in .class files, which is executed by the Java Virtual Machine (JVM).

- It is not Java source code
- It is not machine code
- It is meant to be executed by the JVM
